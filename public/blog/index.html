Our application stack

Posted on 16th of October 2013

I will introduce all the 3rd party products, frameworks and programming languages that are in use at BiblioFair. Main aim was simplicity, speed of development and use of open source tools. So far I do not consider myself a too skilled programmer, but I really would like to contribute back to some of the awesome open source frameworks that make my job easier every day. As I didn't have any prior experience except with static websites, I could aim for the newest high-end tools.

The backbone of the site is the MEAN stack, which stands for MongoDb, ExpressJS, AngularJS, NodeJS. As you might see there is a lot of JavaScript in there.

  • NodeJS is a Chrome's JavaScript runtime for back-end. I could start the server with just about ten lines of code. It runs fast, is easy to use and has a great number of packages to extend functionality. I think it is great for developing web apps, because you can share JavaScript front-end and back-end code. All checks that you make in the browser, usually have to be done on server too, so easy copy-paste is just unbeatable here.
  • ExpressJS is one of the packages for Node. That's the package which makes it so simple to run a server and to route various requests. Listening on a server can be as simple a writing:
    var app = express(); app.listen(port, ipaddr);.
  • MongoDb is a NoSQL database. I can't compare it to a SQL database because I never used one and have an SQL databases course at school right now. For this one I will refer to MongoDB site. To access MongoDB from Node I use MongooseJS package. Even though I had some hard time here, I still think it is a great tool which is evolving very quickly and soon will be a serious competitor to the traditional database businesses.
  • AngularJS is a front-end JavaScript framework extending HTML functionality. It is an open sourced project by Google. There are still things that I have trouble to wrap my head around. I don't know if it's because of me or the bad documentation. But I really like the concept and even though it sometimes takes me a little longer to do some functionality I still think it saved me time overall. Plus, it is getting better as I understand more now.

    • That was the main stack. All design is built upon Bootstrap, a responsive UI front-end framework developed and open sourced by Twitter and our server runs on OpenShift free plan hosting.

    I hope I didn't forget about anything important. If so, leave a comment and I will add the information.
